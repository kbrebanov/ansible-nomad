nomad
=====

[![Build Status](https://travis-ci.org/kbrebanov/ansible-nomad.svg?branch=master)](https://travis-ci.org/kbrebanov/ansible-nomad)

Installs and configures Nomad

Requirements
------------

This role requires Ansible 1.9 or higher.
